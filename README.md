# Segment
![Build badge](https://github.com/segment-dev/segment/actions/workflows/rust.yml/badge.svg)

<img align="right" src="https://www.segment.wtf/img/logo.svg" height="150px" alt="Segment logo">

Segment is a *simple* & *fast* in-memory database with a key-value data model written in Rust.

### Features
- Dynamic keyspaces
- Keyspace level control over eviction
- Multithreaded
